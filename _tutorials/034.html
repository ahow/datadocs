Adding a new scraper (1)

  • Add a script in Scraper scripts
  • Insert records into the Match uploaded columns to scraper inputs-table
  • Insert records into the Scraper variables-table
  • Modify input-file

In order to add a new scraper, you need to do the following:

  • Add a script in the Scraper scripts-table
  • Insert records into the Match uploaded columns to scraper inputs-table (make the link between the excel-file and the python-variable)
  • Insert records into the Scraper variables-table to configure the output-variables
  • Modify the multiscraper-excel-file (add a column and fill it).
