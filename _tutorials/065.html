Manual Running of parts of the script

When you go to the homepage, you click on inputfiles, then you select the file, select the years and start the program.

When you click on Output Last Run Input Data File, you can see the data of the last inputfile. On these data, you can only run one action (Scrape, upper picture).

Before launching the scraping, you might want to modify the scraper-parameters in Input for the Scrapers_Output or in other tables.

The action will launch the scraping in the background and display a progress bar (lower).

At the end of the scraping, you will be redirected to the table Generic Data File. This file can be downloaded in the Admin_Files (column Filename Manual).
