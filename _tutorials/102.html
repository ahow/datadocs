Step 1: scraping input

  • http://scrapecalctool.scibizinformatics.com/admin/base/company/
  • http://scrapecalctool.scibizinformatics.com/admin/base/company_input/: glassdoor URL and eekingalpha

Scraping glassdoor:

  • You need a company_id, this is done by selecting the company in http://scrapecalctool.scibizinformatics.com/admin/base/company/
  • Glassdoor: For that company, you have to make sure that you have a URL in http://scrapecalctool.scibizinformatics.com/admin/base/company_input/, else you won’t scrape anything.
  • Seekingalpha: For that company, you have to make sure that you have a URL in http://scrapecalctool.scibizinformatics.com/admin/base/company_input/, else you won’t scrape anything. In the example above, the URL of apple is used for ABB, so this needs to be corrected still.
