Create input file: Create inputfile (3)

To create the inputfiles: Step 4 export the inputfile (3)

Open Output of the inputfiles, the last record is the file you just exported. Click on the link of the file_name and you have your inputfile. It can be edited or used directly for a normal run (see later). To be loaded for a normal run, you need to rename the file to _Multiple_Webscraper_.xlsx.

The more input-scrapers have run, the better the information in this inputfile.
