Adding a new scraper (4)

  • 1.3. Add a script

    • In order to add a new scraper, you need to do the following:

    • Add a script in the Scraper scripts-table
    • Once the scrape-file has been created, it still needs to be added to the admin-scripts-table
    • Go to admin view, select admin_scripts, click on add_admin_scripts (red in picture above)
    • Give a meaningful name to the script (CSR) and copy-paste the code of steps 1.1 and 1.2 in the script-field
    • Click on save
