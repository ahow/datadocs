Input: Import from Database Tables

Define datapoints

Different tables are used as input for python (and can be modified through the admin views):

Define datapoints:in this table you can define all the datapoints that are stored in the BBG and DS-files. This corresponds to the excel sheet “Matching” form the old Calculations.xlsx-file.

If the datapoint is created before the multiscraper-file is uploaded, this datapoint will be computed.

Based on the code of the first preference (eg BBG for accounts_payable), for each year (eg for 2016), for each isin-code, python will search in the BBG-file (in this case BBG2016.xlsx) for the column BS_ACCT_PAYABLE. This either gives 1 value or no value for that ISIN-code.
