Create input file: Run scrapers

To create the inputfiles: Step 1 Run the scrapers & Step 2 Run fuzzy matching

  • Click on List of scrapers
  • Select the scraper
  • Select the action and click on go. This action removes the words to exclude and compares all the company names of the database with all the names of the company names from the scrapers. This generates a fuzzy score. The best match is copied in the company name of the table
  • You will be redirected to another webpage and a progress bar will show you the progress
  • Two possibilities depending on the value of FUZZY_COMPANY_LIST_FROM_SCRAPER
  • If the value is set to True:
    • Scraper connects to website and scrapes the values and saves the values in the table List of URLs from scrapers for fuzzy matching
    • Once the scraper is done, the fuzzy matching starts
    • After that, you will be redirected to the page with the output of the scraper (Output from fuzzy matching)
  • If the value is set to False:
    • No scraping
    • The fuzzy matching starts using the values from the table List of URLs from scrapers for fuzzy matching
    • After that, you will be redirected to the page with the output of the scraper (Output from fuzzy matching)
  • The fuzzy matching starts automatically after that.
  • Once this is finished, you will be redirected to the output, which you can edit
