Input: using fuzzy string comparison

Then you go to Company sources output (http://scrapecalctool.scibizinformatics.com/admin/base/company_source_scraper/) to see the output.

In the output, you can see the company that fuzzywuzzy thinks matches best with the companies from the source (good guide in this case).

Each company from the scraper (good guide) will be compared to each company in the database, using fuzzy wuzzy.

  • If the fuzzywuzzy-score-value is less than the threshold (determined in step 1 in the constance config), then the company name remains empty and the value remains 0.
  • If the value of the column comp fuzzy name is already filled in, then the value will also not be updated (to avoid that if you correct a value, the next time you run it, it would be overwritten by a wrong value).
  • If the value of the column comp fuzzy name is empty and the fuzzywuzzy-value is higher than the threshold-value, then there will be an update and the comp fuzzy name-value will be filled in and you will see the value of the fuzzy-score.
