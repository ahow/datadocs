Step 2: Preparing Cultural Analysis

  • You need to make sure the following tables are correct:
  • http://scrapecalctool.scibizinformatics.com/admin/base/glassdoorreview/ (output of step 1)
  • http://scrapecalctool.scibizinformatics.com/admin/base/similarityword/ the similarity words need to be selected. If none are selected, the analysis will not start.
  • http://scrapecalctool.scibizinformatics.com/admin/base/similaritymethod/ the similarity method needs to be defined. If none are selected, the analysis will not start.
