Output files

If you want to know if a file finished (status should be different from ‘STARTED’), you have to go to Output files.

In this table, you can find the information relevant to each file: where to find

  • the filename,
  • the outputfile for the input,
  • the outputfile for the scraper
  • the outputfile for the DS/BBG-datapoint-files
  • the outputfile for the metrics
  • The outputfile for the manual run

The last 5 items will be explained later when I write about the manual run. By clicking on any of the links, you can download/open the files.

You can also find the status here, the time of execution, etc.

By clicking on the logfile, you can see what went wrong and what was correct or where the load failed
