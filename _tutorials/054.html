Selection of the outputvariables

At the end of a normal run, you will be redirected to this URL, which allows you to select which scraper-outputvariables, which BBG/DS-outputvariables and which metrics-outputvariables you want to export.

You can also define the formatting for each column. By default, the formatting defined in the ‘define metrics’, ‘define datapoints’ and Scraper variables-tables are used. At this moment, it can be modified still during the run.

You can use actions to check or uncheck several items at the same time.

This is also where you can export and use the simple_download-export.
