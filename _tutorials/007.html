Import companies

There are many ways to insert companies in the database:

  • Manually, by selecting company (add) and filling in the data
  • Using the _Multiscraper_.xlsx-file, see later
  • But the most normal way to insert companies is to use an excel file.
    • Click on Company list
    • If the actions are not visible (because there are no records in the table), create a dummy record.
    • Select the action “Import the file with the companies.” (The same can be achieved using the button in the upper right corner)
    • You will be redirected to another url
    • Browse for the file
    • Upload the file
    • Example of the file in the presentation
    • Once the file has been uploaded, it will be saved in the table Output of the inputfile
  • There is no validation on the file, so it might crash if the formatting is not correct.
