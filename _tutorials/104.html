Step 1: Start Scraping Seeking Alpha

Go to http://scrapecalctool.scibizinformatics.com/admin/base/company/, select 1 or more company/ies. Then you select the action “Scrape seekingalpha for the selected companies”. You will not see any progressbar, but in the background a task has been started that scrapes all the transcripts of seekingalpha.
