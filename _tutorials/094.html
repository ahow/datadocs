Create a scraper like EICC (step 1)

  • Create a scraper in
    http://scrapecalctool.scibizinformatics.com/admin/base/input_scrapers_output/
  • Set all the fields correctly as described below
  • Run the new scraper with few companies and only this scraper selected

Step 2: create record in http://scrapecalctool.scibizinformatics.com/admin/base/input_scrapers_output/
Scraper output name should start with SCR_O, eg SCR_O_AAA_output
Description can be anything you want
fk_scraper has to be the script you created in step 1
Scraper_abbr should be equal to the code in the line "class AAA_Scraper(object):" in this case "AAA"
Class_name should be equal to the code in the line "class AAA_Scraper(object):" in this case "AAA_Scraper"
Function should be equal to the code in the line "def AAA_Scrape(self, company_name, task_id, log_file, teller):" in this case "AAA_Scrape"
Function outputfield: should be equal to the code in the line "return {'AAA_output':AAA_output}" in this case "AAA_output" (the first one)
The 4 following fields can remain as they are
Input1 = SCR_I_company This corresponds to line "def AAA_Scrape(self, company_name, task_id, log_file, teller):". the 3 last parameters are added for logging purposes, even though they are not used in this script, they will be automatically added by the main program, but they don't need to be set in this table.
There is only 1 real input-parameter, so the 4 following input-variables can be left blank
Show-scraper has to be checked if you want to use it
defaultvalue has to be set to 'N' (with the quotes)
Output datatype: has to be set to TEXT
The rest of the table is not relevant for this variable

When you run the main program, it will now check if the company is in the company list scraped from the website
