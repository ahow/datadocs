DP not based on scrapers and not based on BBG/DS

There is a possibility to add datapoints to the output, that are not based on scrapers and not based on BBG/DS-datapoints:

In the _Multiple_Webscraper_.xlsx-file, add a sheet with the following specifications:

  • The sheetname should start with “DP_”
  • The columns should be ISIN, year (this one is optional), name of the output-variable
  • There can be several output-variables for 1 sheet
  • The datatype can vary too : text (string) or number (integer/decimal
  • Example with a year (left) and without (right)

The program, when it imports,

These datapoints are created after the scrapers have run and just after the datapoints from BBG/DS have been computed.

As they are inserted before the metrics are computed, you can use the name of the variables in the metric-definition table and they will be used as if they already existed before the load.
