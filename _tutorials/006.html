Input: Import inputfiles

The input of the python program consists of several types:

  • Inputfiles: _Multiple_Webscraper_.xlsx has the input-information for all the scrapers and should have the following sheets and columns
    • Input
      • ISIN
      • Company
      • BBG Code
      • DS Code
      • Sector
      • Input-parameters for the different scrapers
  • Any number of sheets starting with “DP_”: these are datapoints, whose source is not DS nor BBG
    • These sheets should start with “DP_”
    • The first column in the sheet should be “ISIN”
    • There are two options for the second column
      • Either there is a “year” field
        • In that case the datapoint will be filled in for all the years
      • Or there is no “year” field
        • In this case the datapoint will only be filled in for the current year
  • The file _Multiple_Webscraper_.xlsx will be imported by the user at the start of the run.
  • BBG_2010.xlsx – BBG_2016.xlsx and DS_2010.xlsx-DS_2016.xlsx: These files are stored on the server in the input-directory and will be imported at each run. No action is required from the user to import them.
  • There is a possibility to import companies using an excel file
  • Scraper-specific inputfiles:
    • There is a possibility to import invest iq data from their pdfs using an excel file
    • RTB-input
