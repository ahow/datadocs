Adding a new scraper (6)

  • 2.1. Insert records into the Match uploaded columns to scraper inputs -table

    • In order to add a new scraper, you need to do the following:

    • Insert records into the Match uploaded columns to scraper inputs-table (make the link between the excel-file and the python-variable)
    • Go to the admin views and click Match uploaded columns to scraper inputs-table
    • Click ADD
    • Select the scraper from the dropdown (CSR)
    • Type a new scraper-name-inputname (without spaces, starting with SCR_I_)
    • Copy-Paste the name from the excel-inputcolumn
    • Click on save
