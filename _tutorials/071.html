After using the export-action
Modify the selection-profile

After using the export-action, you should also rename the profile to make sure it has a meaningful name.

To do this, you go back to the admin-page. You select Input_all_profiles. You will notice that it is not possible to add a new profile in the normal way. This is normal, it has been disabled to avoid that you have profiles without any profile_details. To add a profile, you can select one and copy it, using actions. Or you modify one, the same way.

The description is editable, so you should give it a good description. By default it is just “copy of _all”
