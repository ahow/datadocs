Steps of cultural analysis

  • Scrape glassdoor
  • Scrape seekingalpha
  • Run cultural analysis

The cultural analysis is done in two steps:

  • Glassdoor is scraped: all the reviews are scraped: the pros, cons, advices to the management, etc are scraped from the website. The input of this scraper is the company_name/company_id and of course you also need an URL for glassdoor.
  • Seeking alpha is scraped: all the transcripts are scraped from the website. The input of this scraper is the company_name/company_id and of course you also need an URL for seekingalpha.
  • Once the data are in the database, you can start running the cultural_analysis. For this you need to select the words you want to compare, the method you want to apply.
