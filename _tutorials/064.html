Normal Run

When you go to the homepage, you click on inputfiles, then you select the file, select the years and start the program.

After the importing of the files, the data from the inputfile will be saved in the table Output Last Run Input Data File and exported to an excel file. The name of the excel file can be seen in the table Admin_Files (column Filename Input). By clicking on the name of the file, you can download it or just open it.

After the scraping, the data will be saved in the table Output Last Run Scrapers Data File and exported to an excel file. The name of the excel file can be seen in the table Admin_Files (column Filename Scraper). By clicking on the name of the file, you can download it or just open it.

After the BBG/DS-datapoints-calculations, the data will be saved in the table Output Last Run Datapoints Data File and exported to an excel file. The name of the excel file can be seen in the table Admin_Files (column Filename Datapoint). By clicking on the name of the file, you can download it or just open it.

After the metrics-calculations, the data will be saved in the table Output Last Run Metrics Data File and exported to an excel file. The name of the excel file can be seen in the table Admin_Files (column Filename Metric). By clicking on the name of the file, you can download it or just open it.

When the load is finished, the data will be displayed in Output Last Run - Exported file and exported to an excel file. The name of the excel file can be seen in the table Admin_Files (column Filename). By clicking on the name of the file, you can download it or just open it.

The column Filename Manual in Admin_Files remains empty.
