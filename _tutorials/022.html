Create input file

  • Run scrapers to obtain URLs and Run fuzzy matching
  • Edit output of fuzzy matching
  • Run actions to make sure the database is up to date
  • Export the results to excel

Once the companies have been imported, it is possible to start the creation of the inputfile.

This happens in several steps:

  • You need to run the different scrapers to obtain the URLs
  • Run the fuzzy matching
  • Edit the output of the fuzzy matching
  • Export the results to excel
