Deleting a scraper

  • Go to Scraper scripts
  • Select the scraper
  • Click on the action delete selected record

Warning: this will delete all the child-record too.

In order to add a new scraper, you need to do the following:

  • Go to Scraper scripts
  • Select the scraper
  • Click on the action delete selected record

Warning: this will delete all the child-record too.
