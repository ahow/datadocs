http://scrapecalctool.scibizinformatics.com/output/

Connecting directly to http://scrapecalctool.scibizinformatics.com/output/ you get the following screen, where you can select the Sectors, the Industries, the Regions, the years, the metrics and the datapoints you want to export.

Sectors, industries, region, years filter per row.
The metrics and datapoints filter per column.

Another special aspect about this, is that there are no “region and industry”-columns in the table, so I used the isin codes to filter. Eg Nestle has an isin-code. If that isin code is in both the regions and the industries that the user selected, that record is kept, else not.

Once the user clicks on download, he gets an open/save-pop up window.

There is now a possibility to use a filter to search the company or metrics and select them or deselect them.

The “(de)select all”-link allows you to select all the items and pass them to the left or right side.
